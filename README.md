# Skin Disease Classification from Images

An AI-powered web application that classifies skin diseases from dermoscopic images using a Convolutional Neural Network (CNN). The system enables users to upload an image and receive real-time predictions with confidence scores.

---

## 🚀 Features
- Multi-class skin disease classification
- Real-time image prediction
- CNN-based deep learning model
- Simple and interactive Flask web interface
